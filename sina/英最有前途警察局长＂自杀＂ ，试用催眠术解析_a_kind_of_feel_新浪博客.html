英最有前途警察局长＂自杀＂ ，试用催眠术解析

(2009-09-25 03:55:19)
转载▼
标签：

杂谈

 分类： 催眠术
 

英最有前途警察局长"自杀" 美CIA虐囚黑幕或为死亡原因
　　
　　
　　
　　 英国第二大警察局大曼彻斯特警察局局长米切尔·托德11日下午被发现死于英国威尔士北部斯诺登峰。据英国媒体和官方称，托德是跳崖自杀身亡，但也有人怀疑他死于国家阴谋，不能排除他杀的可能性。
　　
　　 民众心目中的英雄
　　 托德是英国最有名的警察，是许多民众心目中的英雄。托德于1976年开始警察生涯，2002年开始出任大曼彻斯特警察局长。
　　
　　 托德因为将更多的警力投入一线确保市民安全，努力将曼彻斯特打造成“一座令人惊奇、绝对安全感的城市”而获广大市民的拥护。
　　
　　 与天天呆在豪华办公室里发号施令的其他高级警官不同，托德经常与特警上街巡逻，亲手实施抓捕。曼彻斯特公民因此赠他“公安先生”的美誉。
　　
　　
　　
　　 纵身跳下悬崖
　　 据托德的副手戴夫·瓦顿11日介绍：“昨天，我们的长官托德请假去威尔士西北部的斯诺登峰‘散散心’。这期间，他分别给几个同事打电话，说自己的精神状态不太好，令接听电话的同事们都觉得有点怪异。昨晚，几个同事碰头交流后，立即对托德的状况感到担心。在打他手机联系未果的情况下，我们紧急动员搜索与救援人员寻找托德。搜寻行动持续到今天下午，我们得到非常不幸的消息……”
　　
　　
　　
　　 政府定性“自杀”
　　
　　
　　
　　 托德死讯证实后，英国政府迅速定性他是“不堪心理重压跳崖自杀”。
　　
　　 一名官员透过媒体说：“他自杀前不是打电话给好几个朋友谈过他的心理状况嘛？大家就觉得他有自杀的倾向。另外，在他遗体边上也发现了一些东西，包括写给他亲友的多张自杀便笺。”
　　
　　
　　
　　 “自杀说”存三大疑团
　　 然而，这种说法立即遭到许多了解托德的人的强烈质疑。首先，托德的家庭稳定，50岁的托德是三个孩子的父亲，尽管他与妻子以及三个孩子两地分居，夫妻之间的感情曾经闹过危机，但最近刚与妻子度假回来，夫妻关系有好转；其次，托德无任何丑闻和不良记录；第三，托德有着光明的前程：他是公认的英国警方第一把交椅的最佳候选人。
　　
　　
　　
　　 ●猜测 虐囚“帮凶”良心不安
　　 托德是美国中央情报局(CIA)虐囚黑幕与英国关系调查的主要负责人。托德曾率特别调查小组对CIA利用英国转运“恐怖嫌犯”，结果导致许多“恐怖嫌犯”遭美国看守虐待一事展开调查。
　　
　　
　　
　　尽管去年6月这一调查的结果显示，“没有证据”证明英国政府在美国特工虐囚丑闻中起作用，但今年2月，英国政府被迫承认，英国确实允许美国CIA借用其在印度洋的“前哨机场”转运“恐怖嫌犯”。
　　
　　
　　
　　导致英国政府如此背运的原因据说是：一名“级别相当高的警官”因不堪良心与职业道德重负而向媒体透露了这一惊人的内幕。外界怀疑这名警官其实就是托德。一旦这一怀疑得到证实，那么托德“跳崖自杀”被外界视为“国家阴谋”就是合乎逻辑的。
　　
　　_________________________________________________________
以下为李扬利用催眠术分析此自杀事件：
　　
　　
　　 1，大家都知道美国非法转移关押恐怖嫌疑人，也都知道欧洲一些国家参与此事
　　在这种情况下，美国铁杆盟友英国参与此事，好象顺理成章，对吧？！
　　
　　2，既然大家都怀疑英国参与此事，那么调查出真实结果，不会出人意料
　　在这种情况下，暗杀调查此事的警察长，好象因小失大
　　
　　3，如果此警长真的死于暗杀，那么只有一种可能：他向媒体揭发了此事，导致英国政府恼怒，盛怒之下，轻意结果了一个人的生命
　　
　　4，如果此案是暗杀，那么此警察长应该死于催眠术
　　
　　5，托德请假去威尔士西北部的斯诺登峰‘散散心’，此事可以催眠诱导所为，即催眠状态下的警长，接受了催眠指令，去山区散心，但他本人不知道这是催眠的结果
　　
　　6，“这期间，他分别给几个同事打电话，说自己的精神状态不太好，令接听电话的同事们都觉得有点怪异”
　　这个情绪状态不好，可以是催眠诱导的结果，如果情报人员下了催眠指令，让他给同事们打电话，说自己心情不好，那么清醒后的警察长会接受潜意识的命令，这样做的！
　　
　　7，“在他遗体边上也发现了一些东西，包括写给他亲友的多张自杀便笺。”
　　情报组织做案时，常常利用自杀这一方法，因为有亲笔书写的自杀遗嘱，所以警察局和社会无法调查下去了！
　　在以前催眠警察长时，让他写下遗嘱，然后催眠指令他在某天某时来到山崖，现场有人催眠他后命令他跳下山崖，接着在现场放下这些自杀遗嘱即可，这种技术很简单和原始！！！
　　
　　关健的问题是，这个英国警察长在出外旅游时，尤其他来到山崖边时，他的身旁是否有人？！这个很重要，如果警察长一个人来到山崖旁，则他是自杀，如果现场和目击证人，都能证明警察长旁边有过人，那么警察长百分之百是被情报组织利用催眠术暗杀的！！！
　　
　　
　　可是李扬一直有个疑问，就为这点事情暗杀一个警察长，值得么。。。
